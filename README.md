# ProProject: The Ultimate Arcade Ecosystem

A high-fidelity, AI-verified competitive gaming platform designed for the elite console community. ProProject is the "Single Source of Truth" for rankings, rivalries, and organized tournaments.

---

## 🚀 The Vision
ProProject transforms the casual console experience into a high-stakes, professional arena. By utilizing **Gemini AI Vision**, we eliminate the need for manual reporting and human error. If it happened on your screen, our AI Oracle knows.

---

## 🏗️ Core Tech Stack

| Layer | Technology |
|-------|-----------|
| **Framework** | Next.js 16 (App Router, React 19, Server Components) |
| **Styling** | Tailwind CSS v4 + Framer Motion (High-Fidelity "Arcade" Theme) |
| **Database** | MongoDB + Mongoose (Strict TypeScript, Atomic Transactions) |
| **Authentication** | NextAuth.js v4 (JWT-based, Secure Role Management) |
| **AI Oracle** | Gemini 1.5 Pro & Flash (Vision, Analysis, and Narration) |
| **Real-time** | Server-Sent Events (SSE) + MongoDB Change Streams |
| **Storage** | AWS S3 with Presigned URL Uploads |
| **Testing** | Playwright (E2E) + Vitest (Unit/Integration) |

## 💎 Legendary Features

### 1. 🛡️ AI Oracle Verification
*   **Screenshot Analysis**: Automatically extracts winners, scores, and methods from UFC 6 (and other sectors) using Gemini Vision.
*   **Video Proof**: Support for YouTube and Twitch clip integration in the Community Tribunal.
*   **Bio-Code Validation**: A unique protocol that ensures users actually own their claimed PSN/Xbox IDs.

### 2. 🏆 Competitive Engine
*   **Dynamic Leaderboard**: Global ELO rankings with "Verified Operator" status badges.
*   **Tournament Brackets**: Automated single-elimination championships with real-time bracket advancement.
*   **Nemesis Watch**: Deep rivalry tracking with head-to-head scorecards and the "Sector Belt" mechanic.

### 3. 💰 High-Stakes Economy
*   **Wager Matches**: Escrow-based credit betting on individual challenges.
*   **Prize Pools**: Tournament entry fees automatically build the Grand Finals pot.
*   **Side-Betting**: The community can bet credits on high-stakes matches using AI-generated "Oracle Odds."

### 4. 🎭 Immersive Social Hub
*   **Real-time Global Chat**: A high-energy lobby for trash talk, recruitment, and sector alerts.
*   **The Narrator**: A 90s arcade-style AI announcer that reacts to every verified victory.
*   **Global Activity Ticker**: A live feed of every challenge, achievement, and victory in the sector.

## 🛠️ Getting Started

1.  **Clone & Install**: `npm install`
2.  **Configure Environment**: Copy `.env.example` to `.env.local` and add your `MONGODB_URI`, `GEMINI_API_KEY`, and AWS credentials.
3.  **Seed the Sector**: `npm run db:seed` to populate the leaderboard and initialize the game protocols.
4.  **Launch**: `npm run dev`
